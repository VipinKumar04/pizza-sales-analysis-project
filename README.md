# 🍕 Pizza Sales SQL Analysis Project

## 📌 Project Overview

This project is a complete SQL-based analysis of a Pizza Sales Database. It contains database schema creation, basic to advanced SQL queries, and datasets used for performing sales analysis.

The project is designed for beginners and intermediate learners who want to practice:

* Database creation
* SQL joins
* Aggregate functions
* Grouping and sorting
* Subqueries
* Window functions
* Common Table Expressions (CTEs)
* Revenue analysis
* Sales reporting

---

## 📂 Project Structure

```bash
├── schema.sql
├── basic query.sql
├── intermediate query.sql
├── advance query.sql
└── pizza data.zip
```

### Files Description

| File Name                | Description                                          |
| ------------------------ | ---------------------------------------------------- |
| `schema.sql`             | Contains table creation and relationships            |
| `basic query.sql`        | Beginner-level SQL queries                           |
| `intermediate query.sql` | Intermediate SQL analysis queries                    |
| `advance query.sql`      | Advanced SQL queries using CTEs and Window Functions |
| `pizza data.zip`         | Dataset files used in the project                    |

---

## 🗄️ Database Schema

The database contains the following tables:

### 1. `orders`

Stores order date and time.

### 2. `order_details`

Stores pizza order details and quantities.

### 3. `pizzas`

Contains pizza size and price information.

### 4. `pizza_type`

Contains pizza names, categories, and ingredients.

## ⚙️ How to Run the Project

### Step 1: Create Database

Create a database in MySQL/PostgreSQL.

### Step 2: Run Schema File

Execute:

```sql
schema.sql
```

### Step 3: Import Dataset

Import CSV files from `pizza data.zip` into the tables.

### Step 4: Execute Query Files

Run:

```sql
basic query.sql
intermediate query.sql
advance query.sql
```
